# 插件概述
WordPress替换前端字体插件是一款针对Wordpress站点开发的一款前端字体替换插件。利用该插件，站长可以快速实现替换站点前端字体等，目前已收录优设标题黑、文道潮黑、仓耳渔阳体-W05、沐瑶软笔手写体、杨任东竹石体等字体。<br>目的在于进一步提升网站的整体美观，让网站更具美感。
