<strong>实现替换使用Wordpress的站点前端字体

== Description ==

Wordpress替换前端字体插件 由<a href="https://blog.itzhiyin.com>宁子</a>开发，实现替换使用Wordpress的站点前端字体。


== Installation ==

1. 上传 `zibll_Font`目录 到 `/wp-content/plugins/` 目录
2. 激活插件，开始设置使用。
